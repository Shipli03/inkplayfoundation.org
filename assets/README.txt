InkPlay Foundation HTML Website Package

Contents:
- index.html
- styles.css
- assets/ (images extracted from the uploaded PowerPoint deck)

How to use:
1. Unzip the package.
2. Open index.html in any modern browser.
3. Upload the complete folder to your web host if publishing online.

Design notes:
- Preserves the uploaded deck's green/cream/tan visual language.
- Refines typography and spacing to reduce clutter.
- Keeps the original section order and expert details.
- Self-contained package with local assets and no external font or script dependency.
